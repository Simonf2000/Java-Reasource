    # 2.3 html的语法规则

    • 根标签是<html></html>有且只能有一个

    • 无论是双标签还是单标签都需要正确关闭；

      • 单标签可以不关(功能上没有影响)

    • 标签可以嵌套但不能交叉嵌套；

    • 注释语法为 ，注意不能嵌套；

    • 属性必须有值，值必须加引号，H5中属性名和值相同时可以省略属性值

    • HTML中不严格区分字符串使用单双引号

    • HTML标签不严格区分大小写，但是不能大小写混用；

    • HTML中不允许自定义标签名，强行自定义则无效；
